== Description ==


Contecta formularios de Contact Form 7 con Mis Leads. 
Ficheros de idiomas disponibles: Ingles, Castellano y Catalán. 

== Installation ==

1. Subir el plugin a la carpeta plugins, o instalarlo usando la opción de Añadir nuevo plugin de Wordpress.
1. Activar el plugin
1. Ir a la página de configuración (Mis Leads)
1. Rellena el cliente y el id fuente
1. Establece los campos de contact form y su equivalencia en Mis Leads
1. Añade el ID Curso a las paginas dónde tengas los formularios de Contact Form 7
1. Guardar cambios.
